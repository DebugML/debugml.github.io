> Large language models (LLMs) often make reasoning errors.
> We introduce a framework that can reliably certify errors with probabilistic guarantees, basing its decision solely on previously-verified premises.
> We find that our framework can reliably detect propagated errors in long reasoning chains, which other methods fail to.

Large Language Models (LLMs) often produce reasoning chains with errors that propagate, undermining the final outputs.
An error can be ungrounded statements, invalid derivations, or propagated errors.
For example, deriving $x=5$ from $5x=9x-20$ is logically valid, but can be a propagated error if the premise $5x=9x-20$ differs from the context.
These errors compromise the reliability LLMs in high-stakes domains.

In this example, <span style='color:orange; font-weight:bold'>Step 3 is Ungrounded</span> because it contains information different from the base claim 2. <span style='color:red; font-weight:bold'>Step 4 is Invalid</span> because it contains an incorrect mathematical computation. <span style='color:#880088; font-weight:bold'>Step 5 is a Propagated error</span>, even though it is logically correct with respect to Step 4." %}


Current error detection methods, such as LLM judges and Process Reward Models (PRMs), typically attempt to identify all errors at once.
However, these methods often fail to detect propagated errors when they are distracted by invalid premises.

To address this issue, we introduce Autoregressive Reasoning Entailment Stability (ARES), a probabilistic framework inspired by human reasoning, which assesses each reasoning step inductively.
It determines a step's soundness by calculating its entailment probability based _only_ on the set of preceding, sound claims, as shown in the figure below.

{% include gallery id="gallery_algo" layout="" caption="**Autoregressive Reasoning Entailment Stability (ARES)** When we verify a reasoning chain, we can break it down into base claims and derived claims. ARES checks each derived claim step-by-step, using previously-verified claims as premise at a probability." %}

Empirically, we find that our method accurately certifies both sound and unsound reasoning steps.
It particularly excels in long chains prone to error propagation, where existing LLM-based approaches fail.

# Detecting Errors via Autoregressive Reasoning Entailment Stability

We first define a reasoning chain as a sequence of base claims $(C_1, \dots, C_n)$ that are given, and derived claims $(C_{n+1},\dots,C_{n+m})$ generated by an LLM.
A probabilistic entailment model $\mathcal{E}(P, H)\mapsto r$ estimates the probability that a premise $P$ entails a hypothesis $H$, where $r\in[0,1]$.

A reasoning chain can be considered as "hard-sound" if every derived claim is perfectly entailed by all preceding claims.
However, this rigid definition is brittle, as a single minor error invalidates the entire chain.
Real-world LLM outputs rarely meet this standard and contain uncertainty.

To address this, we propose Autoregressive Reasoning Entailment Stability (ARES), which computes a stability score $\tau_k$ for each derived claim $C_{n+k}$.
This score represents the expected entailment of $C_{n+k}$ by marginalizing over all $2^{n+k-1}$ possible subsets of valid preceding claims:

$$\tau_{k} = \sum_{\alpha \in \{0,1\}^{n+k-1}} \mathcal{E}(C (\alpha), C_{n+k}) \cdot \Pr[\alpha]$$

where the binary vector $\alpha \in \{0, 1\}^k$ indicates which claims to include ($\alpha_i = 1$) or exclude.

The probability of a premise combination, $\Pr[\alpha]$, is calculated autoregressively.
- For **base claims**, it is the product of their prior soundness probabilities $p_i$: 
$$\Pr[\alpha_{1:n}] = \prod_{i = 1}^{n} p_i ^{\alpha_i} (1 - p_i) ^{\alpha_i}$$
- For **derived claims**, the probability is updated inductively via the chain rule, conditioned on the entailment of the new claim:
$$\Pr[\alpha_{1:n+k}] = \Pr[\alpha_{1:n+k-1}] \,\cdot \quad\, \mathcal{E}(C (\alpha_{1:n+k-1}), C_{n+k})$$

As direct computation is intractable, we estimate $\tau_k$ via efficient Monte Carlo sampling, which comes with a formal statistical guarantee on its accuracy in certifying ARES.

**Theorem 1.** (Ceritifying ARES via Sampling)
Let $N \geq \tfrac{\log(2m/\delta)}{2 \varepsilon^2}$ for any $\varepsilon > 0$ and $\delta > 0$.
Given an entailment model $\mathcal{E}$ and a reasoning chain with $m$ derived claims,
use $N$ i.i.d. samples to estimate each $\tau_k$.
Then, with probability at least $1 - \delta$, we have $|\hat{\tau}_k - \tau_k| \leq \varepsilon$ for all $k$.
{: .notice--info}

The sampling process is illustrated below:

    We first randomly initialize a premise (denoted by $\alpha$) according to the base priors $p_1, \ldots, p_n$.
    Then, for each derived claim, we compute its entailment rate with respect to the premise set.
    Finally, we add this derived claim to the premise set with probability equal to its entailment rate.

# ARES Excels in Long Reasoning Chains with Propagated Errors

We evaluate ARES against baselines like LLM-Judge, ROSCOE, and ReCEval on four benchmarks, including our new synthetic dataset, ClaimTrees, which features long chains with controllable propagated errors.

{% include gallery id="gallery_long_chains" layout="" caption="**(ClaimTrees) GPT-4o-mini.** OURS can robustly identify error propagations in long reasoning chains, whereas other methods fail." %}

Experiments confirm that ARES excels at identifying these propagated errors. As shown in the above figure, ARES maintains a high Macro-F1 score on chains up to 50 steps, while baseline performance deteriorates sharply. This superior performance, illustrated with a concrete example in the following example, is consistent across all datasets.

We attribute this success to ARES being the only method that satisfies all key desiderata for error detection: 
_robust:_ previous errors do not adversely affect the current step, _causal:_ downstream steps do not affect the current step, and _sufficient:_ all relevant claims are included as premise when assessing.

# Conclusion
In this blog post, we show that ARES offers a novel, certifiable, and probabilistic approach to inductively assess reasoning soundness. By considering only previously validated claims, it provides a more principled and reliable method for detecting errors in reasoning chains.

For more details in thoeretical proofs and quantitative experiments, see our [paper](https://arxiv.org/abs/2507.12948) and [code](https://github.com/fallcat/ares).
